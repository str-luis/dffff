# API sencilla de Login con FastAPI

Proyecto mínimo de autenticación con **FastAPI** y **SQLModel**.

## Descripción
Esta API permite validar un usuario y contraseña contra una base de datos SQLite local.
La base se crea automáticamente al iniciar la aplicación y también se inserta un usuario de ejemplo.

## Tecnologías usadas
- FastAPI
- SQLModel
- SQLite
- Uvicorn

## Requisitos
- Python 3.10 o superior
